# Debian Setup Script

This repository contains set of post-installation scripts for Debian ant its derivatives. You can install effortlessly software you want to install after debian installation.

# Install

Firstly, If you do not install sudo, you can use this commands to install sudo

```
git clone https://github.com/ahmetcancicek/debian-post-install.git
cd debian-post-install
su
make .INSTALLSUDO
```

For Debian Post Installation

```
git clone https://github.com/ahmetcancicek/debian-post-install.git
cd debian-post-install
sudo make .INSTALL
```
